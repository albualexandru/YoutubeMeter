# YoutubeMeter

Chrome extension that tracks the total time spent on YouTube each day and shows the running total directly in the extension icon badge.

## Load locally

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select `/tmp/workspace/albualexandru/YoutubeMeter`.

The badge updates automatically and resets at the start of each new day.
